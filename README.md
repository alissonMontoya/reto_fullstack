# 🛍️ E-commerce React

Proyecto desarrollado como parte del curso Full Stack.

## 🚀 Funcionalidades
- Listado de productos
- Carrito de compras
- Navegación entre páginas
- Manejo de estado con Zustand

## 🛠️ Tecnologías
- React
- Vite
- Zustand
- JavaScript

## ▶️ Instalación
npm install
npm run dev
